# Cricket-IQ Project Context

## Product Goal

Cricket-IQ is a mobile cricket intelligence app focused on IPL 2026.

The app provides:
Match schedules
Match details
Win probability predictions
Team insights
Favorite teams

The goal is fast, clean, professional sports app UI similar to 433 football app.

NOT a betting app.
NOT a fantasy app.
NOT a social media app.

Focus is match intelligence.
